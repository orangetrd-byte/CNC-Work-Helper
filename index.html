Lathe Setup Tips

  • Confirm jaws, stickout, stock length, and work offset before touching off.
  • Keep a written note for the exact touch-off surface and tool offset used.
  • Check clearance at both X and Z before the first rapid move.

Touch-off Tips

  • X is diameter based on most CNC lathes, so a .010 diameter change is .005 radial.
  • Z face is usually 0.000. Plunging into the face normally moves Z negative.
  • After touching off, make a short air move before cutting stock.

Safe Approach / Retract

  • Rapid to a safe X and safe Z before feed moves.
  • Retract X clear before moving Z when shoulders, jaws, or grooves are close.
  • Single block the first run and keep feed override low until motion is proven.

Insert / Radius Notes

  • Insert width controls groove width; nose radius affects corner blend and clearance.
  • Confirm programmed path against tool nose compensation and shop standard.
  • For form tools and groovers, check both sides of the insert before calling the move good.

Common Formulas

  • Radius = diameter / 2.
  • Diameter = radius x 2.
  • Lathe radial travel = diameter change / 2.
  • Taper per inch = diameter difference / length.

SFM / RPM

  • RPM = (SFM x 3.82) / diameter.
  • SFM = (RPM x diameter) / 3.82.
  • Use actual stock diameter for starting RPM estimates.

Thread Basics

  • Confirm pitch, hand, class, and major/minor diameter before cutting.
  • Check threading tool angle and center height.
  • Use the control and shop chart for final thread depth.
