PROJECT-ARCHITECTURE.md

Project Name

CNC Work Helper

Purpose

Mobile-first machinist assistant and shop-floor reference tool.

Designed to be useful while standing at a machine.

Primary Focus

- Setup Notes
- Tool Libraries
- G-code Assistance
- G-code Validation
- Toolpath Visualization
- Geometry Calculators
- Speeds & Feeds
- Drill/Tap References
- GD&T References
- Machinist References

Long-Term Vision

Calculator
→ Machinist Helper
→ Setup Assistant
→ G-Code Assistant
→ Shop Floor Companion

Features That Belong Here

- G-code
- Tool Libraries
- Geometry
- Speeds & Feeds
- Reference Charts
- Setup Documentation
- Simulator
- AI Machining Assistant

Features That Do NOT Belong Here

- Travelers
- Work Order Routing
- Capacity Planning
- Production Scheduling
- Machine Dispatching
- Shop Scheduling

Those belong in CNC Cell Planner.

Development Philosophy

Useful in under 30 seconds.

Every feature should help a machinist make a decision at the machine.
