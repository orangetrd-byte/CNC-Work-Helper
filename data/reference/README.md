# CNC Work Helper Reference Data

Split reference package for CNC Lathe Work Helper / CNC Work Helper.

Suggested app use:

- Load `data/reference/index.json` first.
- Fetch each listed JSON file as needed.
- Use these files for reference tabs, symbol lookup, lathe/mill code lookup, 3D printer code lookup, and later quiz/training content.

Files included:

- `metadata.json`
- `index.json`
- `mill-g-codes.json`
- `mill-m-codes.json`
- `lathe-g-codes.json`
- `lathe-m-codes.json`
- `programming-symbols.json`
- `blueprint-gdt-symbols.json`
- `operation-sheet-symbols.json`
- `marlin-3d-printer-g-codes.json`
- `marlin-3d-printer-m-codes.json`

Note: This is a summarized reference. Always verify machine-specific details against the actual Fanuc, Haas, or printer/manual documentation.
